# Maze & Conquest

```text
Pixel-art fantasy maze adventure with quests, customization, and animated world systems
Built to practice large-scale front-end architecture, canvas rendering, and game-style UI flow
```

This repository is a front-end game project where I built a multi-screen browser experience with a custom main menu, hero customization, and a quest-driven level scene.

```text
DISCLAIMER
This is an educational/portfolio game project made for learning and practice.
Not affiliated with any third-party IP or game studio.
All third-party asset rights belong to their respective owners.
```

## What's inside

```text
- Main menu with animated UI panels and modal windows (About + Settings)
- Audio settings (music/SFX toggles + draggable volume sliders)
- Graphics settings toggles (units, trees/bushes, water animations)
- Full customization screen for hero class and faction color
- Animated character sprite preview in Customize
- Canvas-based level scene with tile map rendering and camera zoom logic
- Quest system with trigger tiles, dialogue papers, and progression waypoints
- Notification banner and interaction prompts
```

## Core gameplay flow

```text
1) Start in Main Menu
2) (Optional) Customize your hero class + color
3) Enter level scene
4) Follow waypoints and trigger quest dialogue events
5) Complete objective chain and finish story sequence
```

## Media formats used (learning goal)

```text
PNG    -> Environment, units, buildings, UI, and icon assets
Canvas -> Dynamic scene drawing, overlays, and in-game presentation effects
OGG    -> Music and SFX playback
```

## Built with

```text
HTML5
CSS3
Vanilla JavaScript (modularized by screen/system)
Canvas API (level rendering + UI drawing helpers)
```
